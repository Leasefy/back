Leasify - Panel Inmobiliaria (AGENT)

Flujo del Agente Inmobiliario:
1. Un LANDLORD asigna al agente a una propiedad via POST /properties/:id/agents
2. El agente (rol AGENT) hereda permisos de LANDLORD sobre las propiedades asignadas
3. El agente puede gestionar propiedades, visitas, candidatos, contratos, etc. de las propiedades que tiene asignadas
4. El RolesGuard permite AGENT cuando se requiere LANDLORD, pero el PropertyAccessService valida que el agente tenga acceso a esa propiedad especifica

1. Autenticacion

Nota: El rol AGENT se asigna por un LANDLORD via POST /properties/:propertyId/agents usando el email del agente. El agente debe estar registrado en la plataforma primero (con cualquier rol). Al ser asignado, su rol cambia a AGENT automaticamente.
