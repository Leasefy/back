## v1.3 Requirements

### Subscription Tiers

- [x] **ACCS-01**: Endpoint GET /scoring/:applicationId restringido solo a tenant

- [x] **CRED-03**: Al solicitar evaluacion, puede pagar al momento o descontar de creditos existentes
- [x] **CRED-04**: Historial de transacciones de creditos (compras, usos, saldo)

### Evaluations

- [x] **EVAL-01**: Landlord/inmobiliaria puede solicitar evaluacion unificada via endpoint

- [x] **EVAL-04**: Backend soporta polling del resultado via GET /tenant-scoring/:runId
- [x] **EVAL-05**: Resultado almacenado en DB vinculado a la aplicacion
