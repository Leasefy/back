2. In `registerPayment()` method, AFTER the `prisma.cobro.update()` call (line ~171), add PSE auto-tracking:

```typescript
const updatedCobro = await this.prisma.cobro.update({ ... });

// Auto-track canon for FLEX billing when paid via PSE
if (status === 'PAID' && (dto.paymentMethod === 'PSE' || updatedCobro.paymentMethod === 'PSE')) {
  try {
    await this.flexBillingService.registerPsePayment(
      agencyId,
      updatedCobro.consignacionId,
      updatedCobro.rentAmount,  // 1% on rent canon ONLY, not totalAmount
      updatedCobro.month,
      dto.paymentReference,
    );
  } catch (error) {
    // Log but do NOT block the payment — tracking failure is non-critical
    console.error('[FlexBilling] Failed to auto-track canon:', error);
  }
}

return updatedCobro;
```

CRITICAL: The try/catch ensures CanonTracking failure does NOT roll back the cobro payment. This is a fire-and-complete operation per research anti-patterns.

CRITICAL: Use `updatedCobro.rentAmount` (base rent only), NOT `totalAmount` or `totalWithFees`. The 1% Leasify fee applies only to the rent canon.
