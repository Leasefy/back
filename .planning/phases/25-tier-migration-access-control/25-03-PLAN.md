<objective>
Restrict direct scoring access (GET /scoring/:applicationId and GET /scoring/:applicationId/explanation) to tenant-only, blocking landlords and inmobiliarias from viewing scoring results directly.

Purpose: ACCS-01 requires that only the tenant who owns the application can see scoring directly. Landlords/inmobiliarias will access scoring data through the unified evaluation endpoint (Phase 27). ACCS-02 is satisfied by this same change — blocking direct access forces landlords through the evaluation path.
Output: Scoring controller with tenant-only permission checks on both endpoints.
</objective>
