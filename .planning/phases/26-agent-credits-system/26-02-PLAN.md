  1. **Create `src/agent-credits/agent-credits.service.ts`:**
  Inject `PrismaService` and `PseMockService`. Implement three methods:

  **getBalance(userId: string):**
  - `findUnique({ where: { userId } })` on agentCredit
  - Return `{ balance: credit?.balance ?? 0 }` (0 if no record exists yet)

  **purchaseCredits(userId: string, dto: BuyCreditsDto):**
  - Look up price from `CREDIT_PACK_PRICES[dto.packSize]`. If not found, throw `BadRequestException('Pack de creditos invalido')`
  - Call `this.pseMockService.processPayment(...)` with amount = pack price, description = `Compra de ${dto.packSize} creditos de evaluacion`, and PSE data from dto.psePaymentData
  - If payment status is `'FAILURE'`, throw `BadRequestException('Pago PSE rechazado: ' + paymentResult.message)`
  - For SUCCESS or PENDING (optimistic grant, same pattern as subscriptions):
    - Use `this.prisma.$transaction(async (tx) => { ... })`:
      - `tx.agentCredit.upsert({ where: { userId }, create: { userId, balance: dto.packSize }, update: { balance: { increment: dto.packSize } } })` — store result to get balanceAfter
      - `tx.agentCreditTransaction.create({ data: { userId, type: 'PURCHASE', amount: dto.packSize, balanceAfter: credit.balance, pseTransactionId: paymentResult.transactionId, amountPaidCop: packPrice, description: 'Compra de ${dto.packSize} creditos via PSE' } })`
    - Return `{ balance: credit.balance, creditsAdded: dto.packSize, payment: { transactionId: paymentResult.transactionId, status: paymentResult.status } }`

  **deductCredits(userId: string, amount: number, applicationId: string): Promise<boolean>:**
  - Use `this.prisma.$transaction(async (tx) => { ... })`:
    - Conditional decrement: `tx.agentCredit.updateMany({ where: { userId, balance: { gte: amount } }, data: { balance: { decrement: amount } } })`
    - If `updated.count === 0`, return `false` (insufficient balance)
    - Read updated balance: `tx.agentCredit.findUniqueOrThrow({ where: { userId } })`
    - Create transaction record: `tx.agentCreditTransaction.create({ data: { userId, type: 'DEDUCTION', amount, balanceAfter: credit.balance, applicationId, description: 'Credito de evaluacion usado para solicitud ${applicationId}' } })`
    - Return `true`
  - CRITICAL: Use `updateMany` with `balance: { gte: amount }` for atomicity. Do NOT read-then-write. This prevents double-spend race conditions.
