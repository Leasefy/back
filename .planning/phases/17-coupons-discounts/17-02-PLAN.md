3. Modify the `subscribe()` method:

After calculating `price` from plan config (line that sets `const price = ...`), add coupon handling:

```typescript
let finalPrice = price;
let appliedCouponId: string | null = null;

// Apply coupon if provided
if (dto.couponCode) {
  const validation = await this.couponValidationService.validateCoupon(
    dto.couponCode,
    userId,
    dto.planId,
  );

  if (!validation.valid) {
    throw new BadRequestException(validation.message);
  }

  const coupon = validation.coupon!;
  appliedCouponId = coupon.id;
  const discount = this.couponApplicationService.applyDiscount(finalPrice, coupon);
  finalPrice = discount.finalPrice;

  this.logger.log(
    `Coupon ${coupon.code} applied: original $${price} -> final $${finalPrice} COP`,
  );
}
```

- The PSE required check stays (only if finalPrice > 0)

After EACH subscription creation (both the free/coupon-zero path and the paid path), add coupon usage recording:

// Record coupon usage if applied

IMPORTANT: The recording must happen AFTER the subscription is created so we have the subscription.id. Add it right after the subscription create, before the return statement.

For the free/zero-price path, add the coupon usage recording before the return.
