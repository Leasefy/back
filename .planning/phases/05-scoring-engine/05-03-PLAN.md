1. Create `src/scoring/aggregator/risk-score-result.interface.ts`:
   - Import RiskLevel from '@/common/enums'
   - Export interfaces:
     ```typescript
     export interface Driver {
       text: string;
       positive: boolean;
     }

     export interface Flag {
       code: string;
       severity: 'LOW' | 'MEDIUM' | 'HIGH';
       message: string;
     }

     export interface Condition {
       type: 'DEPOSIT' | 'COSIGNER' | 'INCOME_VERIFICATION' | 'INSURANCE';
       message: string;
       required: boolean;
     }

     export interface RiskScoreResultData {
       total: number;           // 0-100
       level: RiskLevel;        // A, B, C, D
       categories: {
         integrity: number;     // 0-25
         financial: number;     // 0-35
         stability: number;     // 0-25
         history: number;       // 0-15
       };
       drivers: Driver[];       // 3-6 explanations
       flags: Flag[];           // Warning flags
       conditions: Condition[]; // Suggested conditions
     }
     ```

2. Create `src/scoring/aggregator/score-aggregator.ts`:
   - Import { Injectable } from '@nestjs/common'
   - Import RiskLevel from '@/common/enums'
   - Import ModelResult, Signal from models
   - Import RiskScoreResultData, Driver, Flag, Condition

   - Define constants:
     ```typescript
     const LEVEL_THRESHOLDS = {
       A: 80,  // 80-100: Excellent
       B: 65,  // 65-79: Good
       C: 50,  // 50-64: Fair
       D: 0,   // 0-49: Poor
     };
     ```

   - Create @Injectable() ScoreAggregator class with:
     - combine(results: { financial: ModelResult; stability: ModelResult; history: ModelResult; integrity: ModelResult }): RiskScoreResultData

     - Sum all scores:
       ```typescript
       const total = results.financial.score + results.stability.score +
                     results.history.score + results.integrity.score;
       const boundedTotal = Math.max(0, Math.min(100, total));
       ```

     - Calculate level using calculateLevel(boundedTotal)

     - Collect all signals into drivers:
       ```typescript
       const allSignals = [
         ...results.financial.signals,
         ...results.stability.signals,
         ...results.history.signals,
         ...results.integrity.signals,
       ];

       const drivers: Driver[] = allSignals.map(s => ({
         text: s.message,
         positive: s.positive,
       }));
       ```

     - Extract flags from integrity signals (negative ones):
       ```typescript
       const flags: Flag[] = results.integrity.signals
         .filter(s => !s.positive)
         .map(s => ({
           code: s.code,
           severity: this.calculateSeverity(s.weight),
           message: s.message,
         }));
       ```

     - Generate conditions using generateConditions(level, flags)

     - Return RiskScoreResultData

   - Private methods:
     - calculateLevel(score: number): RiskLevel
     - calculateSeverity(weight: number): 'LOW' | 'MEDIUM' | 'HIGH'
     - generateConditions(level: RiskLevel, flags: Flag[]): Condition[]

   - Condition generation logic:
     - Level C: Suggest extra deposit (2 months), not required
     - Level D: Require cosigner
     - Flag 'HIGH_RTI': Require income verification
     - Flag 'UNVERIFIABLE_HIGH_INCOME': Require income verification

       - Build features
       - Calculate all model results
       - Aggregate scores
       - Persist result:
