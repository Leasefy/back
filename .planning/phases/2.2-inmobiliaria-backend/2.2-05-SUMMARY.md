## Decisions Made
- Used @Body with dedicated DTO (GenerateCobrosDto) for POST /generate instead of @Query('month') - better REST semantics for state-changing operations
- cartera-report endpoint calls updateLateFees() first then getSummary() for current month - simple aging report without separate ReportsService
- RegisterPaymentDto changed @IsNumber to @IsInt since Colombian pesos (COP) are integer amounts
- RegisterPaymentDto changed @Matches regex to @IsDateString for paidDate - standard ISO date validation
- Approval endpoint uses both @CurrentAgency('agencyId') and @CurrentUser('id') for agency scoping and audit trail
